## Overview

The Git Repository provides a complete abstraction over Git operations using LibGit2Sharp. It supports commits, branches, remotes, stashing, tagging, and authentication.

**Interface**: `IGitRepository`  
**Implementation**: `LibGit2SharpRepository`  
**Namespace**: `Chapi.Domain.Interfaces` / `Chapi.Infrastructure.Git`

## Commits

### CommitAsync

Creates a new commit with specified files and message.

```csharp
Task<Result<GitCommit>> CommitAsync(
    string projectPath,
    string message,
    IEnumerable<string> files
)
```

<ParamField path="projectPath" type="string" required>
  Path to the Git repository
</ParamField>

<ParamField path="message" type="string" required>
  Commit message
</ParamField>

<ParamField path="files" type="IEnumerable<string>" required>
  Files to stage and commit
</ParamField>

<ResponseField name="GitCommit" type="Result<GitCommit>">
  Returns commit details including Hash, Message, Author, Date, and RelativeDate
</ResponseField>

### GetCommitsAsync

Retrieves commit history with specified limit.

```csharp
Task<IEnumerable<GitCommit>> GetCommitsAsync(
    string projectPath,
    int limit
)
```

<ParamField path="projectPath" type="string" required>
  Path to the Git repository
</ParamField>

<ParamField path="limit" type="int" required>
  Maximum number of commits to retrieve
</ParamField>

<ResponseField name="commits" type="IEnumerable<GitCommit>">
  Collection of commits with Hash, Message, Description, Author, Date, RelativeDate, and Tags
</ResponseField>

### GetUnpushedCommitsAsync

Gets commits that exist locally but haven't been pushed to remote.

```csharp
Task<HashSet<string>> GetUnpushedCommitsAsync(
    string projectPath,
    string branch
)
```

<ResponseField name="commits" type="HashSet<string>">
  Set of commit hashes that are ahead of the remote branch
</ResponseField>

    string projectPath,

<ParamField path="force" type="bool" default="false">

<ParamField path="deleteRemote" type="bool" default="false">

</ParamField>

### ResetAsync

Resets current branch to a specific commit.

```csharp

    string projectPath,
    string target,
    ResetMode mode
)
```

<ParamField path="mode" type="ResetMode" required>

</ParamField>
